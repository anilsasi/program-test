/* This folder is dedicated to bitwise programs */

Bitwise operators:
Bitwise operators may only apply to below types:
    1.char
    2.short
    3.int
    4.long

c provides 6 bit wise operator:
    1.and(&)
    2.inclusive or(|)
    3.exclusive or(^)
    4.left shift operator(<<)
    5.right shift operator(>>)
    6.unary operator(~)
/*------------------------------------*/
and:result will be 1 if two bits are 1
or:result will be 1 if any of bits are 1
xor:result will be 1 if two bits are different
left_shift:left shifts the number of positions
right_shifts:right shifts the number of positions
~:invert all bits
